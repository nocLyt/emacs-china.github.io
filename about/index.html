关于

这是 emacs 中文用户在 github 上的一个信息交流站，负责搜集、整理、更新、维护关于 emacs 的中文资料，也搜集一些国人写的 emacs 项目，并通过 emacs-china 来进行支援。

如果你对这些事情感兴趣，或者有自己写的好的 elisp 项目，欢迎加入我们，联系地址（电子邮箱）： emacs-china@googlegroups.com
